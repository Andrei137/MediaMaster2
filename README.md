# MediaMaster

## Usage

- Activate Run Pubspec Script
```bash
dart pub global activate rps
```

- Install the dependencies
```bash
rps restore
```
or
```bash
flutter pub get
```

- Run in debug mode
```bash
rps dev
```
or
```bash
flutter run -d windows
```

- Run in release mode
```bash
rps run
```
or 
```bash
flutter run -d windows --release
```
